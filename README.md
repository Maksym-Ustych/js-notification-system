Дисципліна: Основи програмування мовою JavaScript

Опис

У даному проєкті реалізовано систему повідомлень (Notification System) за допомогою JavaScript та DOM-маніпуляцій.

Система дозволяє створювати повідомлення різних типів:

Success

Error

Warning

Info

Кожне повідомлення відображається у вигляді окремого блоку з кольоровим індикатором.

Функціонал

створення повідомлень різних типів

stacking повідомлень (кілька одночасно)

автоматичне закриття повідомлення

progress bar часу відображення

ручне закриття кнопкою ×

кнопка дії (OK)

вибір позиції повідомлень на сторінці
